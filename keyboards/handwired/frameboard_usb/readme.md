# Frameboard USB

![Frameboard USB](https://i.imgur.com/DHzgwa3.jpeg)

A standalone computer keyboard based on the [Framework](https://frame.work/) Laptop 13 [keyboard matrix](https://frame.work/products/keyboard?v=FRANBKEN05). [More info in its dedicated GitHub repo](https://github.com/xjanu/frameboard-usb)

* Keyboard Maintainer: [Martin Jan≈Ø](https://github.com/xjanu)
* Hardware Availability: This is a custom project, check the [GitHub repo](https://github.com/xjanu/frameboard-usb)

Make example for this keyboard (after setting up your build environment):

    make handwired/frameboard_usb:default

Flashing example for this keyboard:

    make handwired/frameboard_usb:default:flash

See the [build environment setup](getting_started_build_tools) and the [make instructions](getting_started_make_guide) for more information. Brand new to QMK? Start with our [Complete Newbs Guide](newbs).

## Bootloader

Enter the bootloader using bootmagic reset: hold down the top left (Escape) key and plug in the keyboard.
